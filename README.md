# DevOps_Labs1

## Описание

`DevOps_Labs1` - это проект, включающий реализацию калькулятора, который может выполнять базовые арифметические операции, такие как сложение, вычитание, умножение и деление, а также поддерживает работу с унарным минусом и скобками.

## Структура проекта

- **Calculator.cs**: Основной файл, содержащий реализацию класса `Calculator`, который выполняет разбор и вычисление арифметических выражений.
- **Token.cs**: Файл, содержащий структуру `Token`, которая представляет числа и операции в выражении.
- **Operation.cs**: Файл, содержащий перечисление `Operation`, которое определяет возможные операции (сложение, вычитание, умножение, деление, унарный минус, открывающая и закрывающая скобки).
- **TestConsoleCalculate.cs**: Файл с тестами для проверки корректности работы методов класса `Calculator`.
